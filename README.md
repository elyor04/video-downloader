# Video Downloader
Download videos or audios from YouTube, Instagram, TikTok, and other sites.

## Installation

```
pip install yt-dlp PySide6
```

### Windows
```
winget install ffmpeg
```

### MacOS
```
brew install ffmpeg
```

### Linux
```
sudo apt install ffmpeg
```

## Usage
`python main.py` - Windows <br>
`python3 main.py` - MacOS, Linux
